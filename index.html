Common CSS Property Documentation

Color and Background-color

Property Value Purpose Examples

Named color

color keywords are used to define a color, a list of color keywords can be found here.

color: red;

background-color: red;

hex color

use hexadecimal values to set color

color: #fcf403;

background-color: #fcf403;

rgb color

Set foreground or background color with red green and blue channels

color: rgb(0-255,0-255,0-255);

background-color: rgb(0-255,0-255,0-255);

hsl color

Set foreground or background color with hue, saturation and lightness values

color: hsl(0-360,0-100%,0-100%);

background-color: hsl(0-360, 0-100%, 0-100%);

alpha/opacity

Sets the opacity value of a color, can be used with hex, rgb or hsl

Note: Cannot be used on named colors

color: #fcf403ff;

color: rgba(0-255,0-255,0-255,0-1);

color: hsla(0-360,0-100%,0-100%,0-1);

Box Model Properties

Padding

Property Value Purpose Examples
1 value Sets padding equal on all sides of the content

padding: 10px;
2 values Order: top and bottom, left and right

padding: 10px 30px;
3 values Order: top, right and left, bottom

padding 10px 30px 40px;
4 values Order: top, right, bottom, left

padding 10px 30px 40px 50px;
padding-top Sets padding on the top only

padding-top: 10px;
padding-right Sets padding on the right only

padding-right: 10px;
padding-bottom Sets padding on the bottom only

padding-bottom: 10px;
padding-left Sets padding on the left only

padding-left: 10px;

Border

Property Value Purpose Examples
Shorthand Used to set width, style and colour at once

border: 5px solid red;
border-width Used to set the border width/thickness

border-width: 10px;
border-style Used to set the style of the border (solid, dashed etc.)

border-style: solid;
border-color Used to set the color of the border

border-color: red;
